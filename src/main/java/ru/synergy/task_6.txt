Урок 6.ApacheHttp
Цель задания: знакомство с веб-сервером apache
Задание:
1. Что такое Apache в контексте протокола HTTP?
 Apache - это сервер с программным обеспечением — это посредник между сайтами и браузером пользователя.
 Apache имеет в своем ядре протокол HTTP, посредством которого идет обмен информацией между пользователями и  сайтами.
2. Для чего используется Apache в рамках работы протокола HTTP?
 Получив от клиента запрос по HTTP, он его обрабатывает, находит ответ на нужном сайте и отправляет ответ
 обратно клиенту по протоколу HTTP.
3. Существуют ли альтернативы Apache?
Nginx — второе по популярности веб-серверное приложение и главный конкурент Apache
Веб-сервер Lighttpd — кроссплатформенное программное обеспечение на языке С
Internet Information Services (IIS) — набор сервисов для создания веб-сервера от компании Microsoft
Apache Tomcat — это контейнер сервлетов, который обрабатывает спецификации Java.
